Get started

# 1. Install
curl -fsSL https://mcprecon.pages.dev/install.sh | bash

# 2. Authenticate (run once, stored globally)
recon login sk-recon-...

# 3. Index your project + wire up your IDE
cd your-project
recon init --mcp cc

Delete account
Permanently removes your user record, all API keys, sessions, payments, and subscription history. Any active Razorpay subscription is cancelled immediately. This cannot be undone.

Revoke API key?

Any running recon serve using this key will lose access within 15 minutes, and the key can't be used for new logins. This cannot be undone.

Rotating will generate a replacement key immediately after revocation.

Cancel subscription?

You'll keep full access until the end of the current billing period, which is shown on your dashboard. After that, your account drops to the Free tier.

You can resubscribe any time — your existing indexes stay on disk.
